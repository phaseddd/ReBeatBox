<objective>
Create the keyboard-to-MIDI mapping utility that translates AWT KeyEvent keyCodes to MIDI note numbers per the D-01 three-row layout, with boolean[128] active-note tracking for OS auto-repeat deduplication.

Purpose: The `KeyboardMapper` is the pure-logic heart of the live keyboard feature. It implements the exact key-to-note mapping table defined in CONTEXT.md D-01, covering three full octaves (C3-C6) across the number row, QWERTY row, and bottom row of a standard US keyboard. The deduplication array prevents Windows/Linux auto-repeat from flooding the MIDI synthesizer with duplicate NoteOn messages.

Output: `KeyboardMapper.java` (final utility class) and `KeyboardMapperTest.java` (JUnit 5 tests) in the new `com.rebeatbox.live` package.
</objective>

    Class structure (following NoteColorMapper pattern):
    ```java
    package com.rebeatbox.live;

    import java.awt.event.KeyEvent;
    import java.util.HashMap;
    import java.util.Map;

    /**
     * Three-row computer keyboard to MIDI note mapping (D-01).
     * Covers C3-C6 across number row, QWERTY row, and bottom row.
     * Tracks active note state for OS auto-repeat deduplication (D-03).
     */
    public final class KeyboardMapper {

        /** KeyCode (VK_*) to MIDI note number (0-127) lookup. */
        private static final Map<Integer, Integer> KEY_TO_NOTE = new HashMap<>();

        static {
            // Upper octave — number row: C5-C6
            KEY_TO_NOTE.put(KeyEvent.VK_1, 72);        // C5
            KEY_TO_NOTE.put(KeyEvent.VK_2, 73);        // C#5
            KEY_TO_NOTE.put(KeyEvent.VK_3, 74);        // D5
            KEY_TO_NOTE.put(KeyEvent.VK_4, 75);        // D#5
            KEY_TO_NOTE.put(KeyEvent.VK_5, 76);        // E5
            KEY_TO_NOTE.put(KeyEvent.VK_6, 77);        // F5
            KEY_TO_NOTE.put(KeyEvent.VK_7, 78);        // F#5
            KEY_TO_NOTE.put(KeyEvent.VK_8, 79);        // G5
            KEY_TO_NOTE.put(KeyEvent.VK_9, 80);        // G#5
            KEY_TO_NOTE.put(KeyEvent.VK_0, 81);        // A5
            KEY_TO_NOTE.put(KeyEvent.VK_MINUS, 82);    // A#5
            KEY_TO_NOTE.put(KeyEvent.VK_EQUALS, 83);   // B5

            // Middle octave — QWERTY row: C4-C5
            KEY_TO_NOTE.put(KeyEvent.VK_Q, 60);        // C4
            KEY_TO_NOTE.put(KeyEvent.VK_W, 61);        // C#4
            KEY_TO_NOTE.put(KeyEvent.VK_E, 62);        // D4
            KEY_TO_NOTE.put(KeyEvent.VK_R, 63);        // D#4
            KEY_TO_NOTE.put(KeyEvent.VK_T, 64);        // E4
            KEY_TO_NOTE.put(KeyEvent.VK_Y, 65);        // F4
            KEY_TO_NOTE.put(KeyEvent.VK_U, 66);        // F#4
            KEY_TO_NOTE.put(KeyEvent.VK_I, 67);        // G4
            KEY_TO_NOTE.put(KeyEvent.VK_O, 68);        // G#4
            KEY_TO_NOTE.put(KeyEvent.VK_P, 69);        // A4
            KEY_TO_NOTE.put(KeyEvent.VK_OPEN_BRACKET, 70);   // A#4
            KEY_TO_NOTE.put(KeyEvent.VK_CLOSE_BRACKET, 71);  // B4

            // Lower octave — bottom row: C3-C4
            KEY_TO_NOTE.put(KeyEvent.VK_Z, 48);        // C3
            KEY_TO_NOTE.put(KeyEvent.VK_X, 49);        // C#3
            KEY_TO_NOTE.put(KeyEvent.VK_C, 50);        // D3
            KEY_TO_NOTE.put(KeyEvent.VK_V, 51);        // D#3
            KEY_TO_NOTE.put(KeyEvent.VK_B, 52);        // E3
            KEY_TO_NOTE.put(KeyEvent.VK_N, 53);        // F3
            KEY_TO_NOTE.put(KeyEvent.VK_M, 54);        // F#3
            KEY_TO_NOTE.put(KeyEvent.VK_COMMA, 55);    // G3
            KEY_TO_NOTE.put(KeyEvent.VK_PERIOD, 56);   // G#3
            KEY_TO_NOTE.put(KeyEvent.VK_SLASH, 57);    // A3
        }

        /** Active note state for OS auto-repeat deduplication (D-03). */
        private final boolean[] activeNotes = new boolean[128];

        private KeyboardMapper() {
            // Utility class — no instantiation outside package
        }

        /**
         * Looks up the MIDI note for a KeyEvent keyCode.
         * @param keyCode the AWT key code (e.g., KeyEvent.VK_Q)
         * @return MIDI note number (0-127), or -1 if the key is not mapped
         */
        public static int keyCodeToNote(int keyCode) {
            Integer note = KEY_TO_NOTE.get(keyCode);
            return note != null ? note : -1;
        }

        /**
         * Checks whether a MIDI note is currently active (key is held down).
         */
        public boolean isActive(int note) {
            if (note < 0 || note >= 128) return false;
            return activeNotes[note];
        }

        /**
         * Sets the active state of a MIDI note.
         */
        public void setActive(int note, boolean active) {
            if (note < 0 || note >= 128) return;
            activeNotes[note] = active;
        }

        /**
         * Clears all active note states. Called on window focus loss.
         */
        public void clearAll() {
            for (int i = 0; i < 128; i++) {
                activeNotes[i] = false;
            }
        }
    }
    ```

    The constructor is `private` to prevent external instantiation — but the `ReBeatBoxWindow` (in `com.rebeatbox.ui` package) needs to create an instance. So make the constructor package-private (no access modifier) or public. Since `KeyboardMapper` is a utility class with instance state (activeNotes), and ReBeatBoxWindow is in a different package, use a **public constructor**. This is a deliberate exception to the NoteColorMapper utility pattern — KeyboardMapper has per-instance state.
