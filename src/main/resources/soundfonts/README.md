# SoundFont Directory

Place `FluidR3_GM.sf2` here for high-quality MIDI synthesis.

Download from:
- https://archive.org/download/fluidr3-gm-gs/FluidR3_GM.sf2
- https://musical-artifacts.com/artifacts/3

License: MIT
Size: ~141MB

If no SoundFont is present, the app falls back to the JDK's built-in default soundbank.
